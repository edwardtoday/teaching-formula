# 第 04 课：两步方程（`ax + b = c`）（30 分钟）

## 学习目标

1. 会解并解释：`ax + b = c`（以整数解为主）。
2. 知道“解方程就是把 x 单独留下来”，常用顺序：先去掉 `+b/-b`，再去掉 `×a`。
3. 能写出规范步骤，并代回检验。

## 教学流程（30 分钟）

### 0) 热身 3 分钟：复习一步方程

各 1 题口答并检验：
- `x + 6 = 11`
- `5x = 15`

### 1) 新概念 8 分钟：把两步拆成两次“等式不变”

示范题：`3x + 2 = 14`

引导：
1. “我们想得到 `x = ?`，现在 `x` 被什么‘缠住’了？”（+2 和 ×3）
2. “先消掉 +2：两边 -2”
3. “再消掉 ×3：两边 ÷3”

形象比喻（帮助记顺序，但不替代原理）：
- **脱衣服**：先脱“外套”（加减的那层 `+2/-2`），再脱“里面那层”（乘除的那层 `×3/÷3`）。

写成完整步骤（要求学生读出来）：
`3x + 2 = 14`  
两边 -2：`3x = 12`  
两边 ÷3：`x = 4`  
检验：`3×4+2=14`

### 2) 你带着做 12 分钟：让学生说“为什么先这样”

按难度递增（都保证整数解）：
1. `2x + 5 = 17`
2. `4x - 3 = 21`
3. `6x + 0 = 30`
4. `3x + 7 = 7`

每题追问两句：
- “你这一步对两边做了什么？”
- “你为什么先消掉这个，而不是先除？”

### 3) 学生独立 5 分钟：写步骤+检验

- `5x + 1 = 26`
- `2x - 4 = 10`

### 4) 总结 2 分钟：今天的“顺序感”

> 想把 x 单独留下来：先把“加减”去掉，再把“乘除”去掉；每一步两边同做；最后代回检验。

## 退出卡（1 分钟）

1. 口答：解 `3x + 2 = 14` 为什么先两边 `-2`，再两边 `÷3`？
2. 【找茬】有人把 `2x + 6 = 10` 先两边 ÷2，写成 `x + 6 = 5`。他错在哪里？（提示：6 也要 ÷2 吗？）

## 一对一分层策略（快 / 慢分支）

- **慢：** 每题先用“脱衣服”说顺序（外套=加减，里面=乘除），再写符号；每一步都写成完整等式。
- **快：** 加 1 题“多种做法比较”：先减再除 vs 先除再减（会出现分数），让学生说明为什么课堂更推荐“先减后除”。

## 板书设计（建议）

```text
目标：x 单独留下来
3x + 2 = 14
两边 -2 → 3x = 12
两边 ÷3 → x = 4
检验：3×4+2=14 ✓
```
