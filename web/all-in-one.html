天平解方程

目标：把 x 单独留下来。每一步必须是“对两边同时做同样的事”，最后代回检验。

交点 = 解（GeoGebra）

一条线代表一个条件。交点的坐标是同时满足两个条件的那一对 (x, y)。

平方 = 面积

用正方形面积理解 x²。主线只写非负解；拓展可提到 (-x)² 也等于 x²。

建议上课节奏：热身 3min → 新概念 8min → 带做 12min → 独立 5min → 总结 2min
